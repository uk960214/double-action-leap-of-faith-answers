비서데스크

1. 비서의 ID 카드 찾기

1단계

'서랍'을 탐색해서 '서랍 비밀번호 입력' 퀘스트 해제

2단계

'책상'을 탐색해서 단서 '명찰' 획득

3단계

'명찰'을 탐색해서 비사의 이름 '김미서' 획득

4단계

나인에게 비서의 이름을 전달 후 전화번호 뒷자리 얻기

전화번호 뒷자리

1173

5단계

'서랍 비밀번호 입력' 퀘스트에 김미서의 전화번호 뒷자리 입력해서 아이템 '비서의 ID 카드' 획득

2. 저택내부 보안해제 방법 찾기

1단계

'책상'을 탐색해서 단서 '컴퓨터', '탁상 달력'을 획득

2단계

'컴퓨터'를 탐색해서 컴퓨터 비밀번호 힌트 'Main's Birthday' 획득

3단계

'탁상 달력'을 검색해서 메인의 5주년 창립 기념일 단서 획득

4단계

컴퓨터 비밀번호 입력해서 '폴더 잠금 해제하기' 퀘스트를 해제, 단서 '메모지' 획득

컴퓨터 비밀번호를 얻는 방법

탁상 달력의 Main 무역 5주년 기념일 "20220211"에서 5년 전을 계산하면 암호 "20170211"

5단계

'메모지'를 탐색해서 폴더 잠금 비밀번호 획득

폴더 잠금 비밀번호를 얻는 방법

메모지에서 각 글자를 키보드에서 찾아 글자 아래 화살표 한 칸 이동한 글자를 조합하면 암호 'crucial' 획득

6단계

폴더 잠금 해제 퀘스트에 비밀번호 입력해서 단서 '저택 내부 보안 해제 방법' 획득

7단계

'저택 내부 보안 해제 방법'을 탐색해서 비서의 고유 번호 '453763'을 나인에게 전달

접객실

회의실로 이동할 목적 찾기

1단계

'테이블'을 탐색해서 아이템 '리모컨' 획득

2단계

'리모컨 버튼 누르기' 퀘스트에서 meeting을 입력해서 모니터 켜기

3단계

'모니터가 켜졌다!' 퀘스트를 클릭해서 단서 '인터뷰 사진' 획득

4단계

'인터뷰 사진'을 탐색해서 녹음기 위치인 '샹들리에' 발견 후 나인에게 전달

5단계

'이제 접객실은 다 둘러본 것 같다' 퀘스트를 클릭해서 '회의실에 가는 목적' 퀘스트 해제

6단계

나인에게 '비상시스템' 단서 획득 후 '회의실에 가는 목적' 퀘스트에 입력

회의실

1. 안방 출입 암호 힌트 찾기

1단계

'비품함'을 탐색해서 비밀번호 힌트 “SupPLy” 획득

2단계

'비품함 열기' 퀘스트에 비밀번호 '↑↓↓↑↑↓' 입력해서 아이템 'A4 용지' 획득

3단계

'프린터기'를 탐색해서 단서 '견적서' 획득

4단계

'견적서'를 탐색 안방 출입 암호 힌트 “술 제조년도” 획득 후 나인에게 전달

2. 비상시스템 작동시키기

1단계

나인에게서 '분전함' 단서 획득

2단계

'분전함'을 탐색해서 단서 '매뉴얼' 획득

3단계

'매뉴얼'을 탐색해서 TEST 모드 작동 코드 'UTFT' 획득

4단계

'비상시스템 코드 입력' 퀘스트에 작동 코드를 입력해서 '회장실로 진입하기' 퀘스트 해제

3단계
